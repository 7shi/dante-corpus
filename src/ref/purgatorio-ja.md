# Purgatorio scene breakdown

## Canto 1: 煉獄の夜明けとカトーの守護

| 行 | 場面名 | 内容 |
|---|---|---|
| 1–12 | 詩の新出発 | 地獄の海を離れ、煉獄を歌うと宣言し、詩神カリオーペに助けを求める。 |
| 13–27 | 夜明けの空と四つ星 | 地獄の暗闇から出ると澄んだ空気が広がり、東の空には四つの聖なる星が輝く。 |
| 28–48 | カトーの出現と問い | 白髪の老人カトーが現れ、誰が永劫の牢獄を逃れてここに来たのかと厳しく問う。 |
| 49–75 | ウェルギリウスの弁明 | ウェルギリウスが天上の助けを得てダンテを導いていると説明し、カトーに通行を乞う。 |
| 76–84 | マルツィアの願い | ウェルギリウスが旧知マルツィアへの愛を引き合いに出して許しを求め、カトーが天上の命に従うよう告げる。 |
| 85–108 | 浄化の指示 | カトーが葦を帯にして顔を清めるよう命じ、山への道を示してから姿を消す。 |
| 109–136 | 浜辺での浄化 | ウェルギリウスがダンテの顔を露で洗い、浜辺で葦を帯に結ぶと摘んだ跡に新しい葦が生え直る。 |

|---|---|---|

| 121–136 | 王統の衰え | 王家の徳は枝に戻りにくいと総括し、名ばかりの権勢のむなしさを述べる。 |

## Canto 19: 怠慢のP消去と貪欲の圏

|---|---|---|

| 34–69 | 目覚めと天使の誘導 | ウェルギリウスに呼び起こされ、「悲しむ者は幸いなり」と歌う天使に従って第五回廊へ上がる。 |
