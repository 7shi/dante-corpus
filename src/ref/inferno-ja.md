# Inferno scene breakdown

## Canto 1: 暗い森とウェルギリウスの出現

| 行 | 場面名 | 内容 |
|---|---|---|
| 1–12 | 暗い森での迷失 | 人生の正しい道を失い、暗い森にいることに気づく。 |
| 13–30 | 丘と光への希求 | 丘の上の光を見て希望を得て、登ろうとする。 |
| 31–60 | 三匹の獣の妨害 | 豹・獅子・雌狼が現れ、進路をふさぐ。 |
| 61–90 | ウェルギリウスとの出会い | ダンテがウェルギリウスを見いだし、救いを求める。 |
| 91–111 | 雌狼の説明とヴェルトロの予言 | ウェルギリウスが雌狼の本性を語り、これを倒す者の到来を予言する。 |
| 112–129 | 冥界巡歴の提案 | 地獄・煉獄を通る別の道と、その先の導き手について説明する。 |
| 130–136 | 同行の決意と出発 | ダンテが導きを願い、ウェルギリウスに従って進み始める。 |
