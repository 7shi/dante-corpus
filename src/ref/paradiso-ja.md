# Paradiso scene breakdown

## Canto 1: アポロへの祈願と天への上昇

| 行 | 場面名 | 内容 |
|---|---|---|
| 1–12 | 序詞：神の栄光と詩の宣言 | 神の栄光が宇宙全体に満ちていると述べ、最高天で見た事柄を記憶の限りで歌うと宣言する。 |
| 13–36 | アポロへの霊感の祈願 | 最後の大仕事に際してアポロに神的霊感を求め、成就すれば月桂冠で報いると誓う。 |
| 37–48 | ベアトリーチェの太陽凝視 | 春分の夜明けにベアトリーチェが鷹のごとく太陽を凝視する。 |
| 49–75 | 太陽直視とトラスマナール | ダンテがベアトリーチェに倣い太陽を見つめ、人間を超えた変容を体験する。 |
| 76–84 | 天球の調和と強烈な光 | 天球の調和の音と光に包まれ、かつてない鋭い望みが燃え上がる。 |
| 85–93 | ベアトリーチェの最初の言葉 | ベアトリーチェが口を開き、ダンテがまだ地上にいると思い込んでいる誤りを正す。 |
| 94–99 | ダンテの新たな疑問 | 一つ目の疑念は解けたが、なぜ軽い天球を超えて上昇できるのかと新たに問う。 |
| 100–126 | 万物の秩序 | ベアトリーチェが万物は神に似せられた秩序のうちにあり、各被造物は固有の本性に従って定められた行き先へ向かうと説く。 |
| 127–142 | 自然な上昇の理 | 被造物は自由の誤用で本道を外れうるが、本来ダンテの上昇は川が下るように自然な運動だとベアトリーチェが結論づける。 |

| 124–148 | 光の形式的原理 | 星々の明暗の差は密度ではなくそれぞれに宿る徳の違いに由来するとベアトリーチェが結論づける。 |
